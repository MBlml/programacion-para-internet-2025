Flexbox

¿Qué es?

El Módulo de Caja Flexible, comúnmente llamado flexbox, fue diseñado como un modelo unidimensional de layout, y como un método que pueda ayudar a distribuir el espacio entre los ítems de una interfaz y mejorar las capacidades de alineación.

Esta página hace un repaso de las principales características de flexbox, las que exploraremos con mayor detalle en el resto de estas guías.

Cuando describimos a flexbox como unidimensional destacamos el hecho que flexbox maneja el layout en una sola dimensión a la vez ya sea como fila o como columna.

Flex Direction

Flex Direction: Row y Row-Reverse

Si elegimos row o row-reverse, el eje principal correrá a lo largo de la fila según la dirección de la línea horizontal o en reversa.

Flex Direction: Column y Column-Reverse

Al elegir column o column-reverse el eje principal correrá desde el borde superior de la página hasta el final según la dirección del bloque verticalmente o en reversa.

Flex Wrap

Flex Wrap: Wrap y No-Wrap

Con wrap los elementos se envuelven en varias líneas si es necesario, y el contenedor crece en la dirección del eje transversal.
En no-wrap los elementos se colocan en una sola línea, incluso si esto provoca que se desborden del contenedor, es el valor por defecto.

Flex Flow: Row Wrap

Para flex-flow, es una propiedad abreviada que combina dos propiedades: flex-direction y flex-wrap.
Esto permite definir cómo se distribuyen los elementos flexibles dentro de un contenedor en una sola línea de código.

Align Items & Justify Content

Align Items

La propiedad align-items alineará los ítems sobre el eje cruzado.

stretch es el valor inicial, los ítems se ajustan por defecto a la altura del más alto.
center para alinearlos al centro.
flex-start para que los ítems se alineen al comienzo del contenedor flex.
flex-end para alinearlos al final.

Justify Content

La propiedad justify-content es usada para alinear los ítems en el eje principal, cuyo flex-direction define la dirección del flujo.

flex-start es el valor inicial, alineará los ítems al inicio del margen del contenedor.
center para alinearlos al centro.
space-between para tomar todo el espacio sobrante después de que los ítems hayan sido colocados
space-around para crear un espacio equitativo a la derecha e izquierda de cada ítem.

CSS Grid

¿Qué es?

El Módulo de CSS Grid Layout, comúnmente llamado Grid, es un sistema de diseño bidimensional que permite a los desarrolladores web crear diseños complejos y responsivos de manera sencilla.

Esta página hace un repaso de las principales características de Grid, las que exploraremos con mayor detalle en el resto de estas guías.

Grid permite trabajar con filas y columnas simultáneamente, lo que facilita la creación de diseños complejos.

Grid Template

Grid Template Columns

La propiedad grid-template-columns define las columnas de la cuadrícula. Puedes especificar el tamaño de cada columna utilizando unidades como px, %, fr, etc.

Grid Template Rows

La propiedad grid-template-rows define las filas de la cuadrícula. Similar a las columnas, puedes especificar el tamaño de cada fila utilizando diferentes unidades.

Más Ejemplos de Grid

Grid Auto Flow

La propiedad grid-auto-flow controla cómo se colocan automáticamente los elementos en la cuadrícula. Puede ser en filas o columnas.

Grid Area

La propiedad grid-area permite asignar un nombre a un área específica de la cuadrícula, que luego puede ser referenciada en grid-template-areas.

Grid Template Areas

La propiedad grid-template-areas permite definir un diseño de cuadrícula utilizando nombres de áreas, lo que facilita la organización de los elementos.

Grid Auto Columns

La propiedad grid-auto-columns define el tamaño de las columnas generadas automáticamente en una cuadrícula.
